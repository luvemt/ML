# ML
There are 2 different program; the first program runs the binary classifiers and the other program runs the multi-class classifier
To run the binary classifiers, type:
    python main.py <classifier> <dataset>
    e.g python main.py perceptron iris.data
 
To run the multi-class classifier, type:
    python multiclass.py <dataset>
    e.g python multiclass.py iris.data
